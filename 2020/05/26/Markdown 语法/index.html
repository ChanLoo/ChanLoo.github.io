Markdown 语法

Markdown 语法

字体

加粗 **word**

倾斜 *word* _word_

倾斜加粗 ***word***

划线 ~~word~~

链接

插入图片

![图片描述](图片路径)

自动链接

<https://www.google.com>

分割线

一行中用三个以上的 *, -, _ 来建立分割线

代码块

行内式只要用反引号`就好

多行代码块使用三个反引号包裹

在代码区块里面， & 、 < 和 > 会自动转成 HTML 实体，这样的方式让你非常容易使用 Markdown 插入范例用的 HTML 原始码，只需要复制贴上，剩下的 Markdown 都会帮你处理。

引用

在被引用文本前加上 > 符号，以及一个空格

Word

Word

Word

Word

Word

引用的区块内也可以使用其他的 Markdown 语法，包括标题、列表、代码区块等。

Code

列表

无序列表

使用 *, +, -

  • First
  • Second

有序列表

使用数字和一个英文句点表示有序列表。

  1. 一
  2. 二

在使用列表时，只要是数字后面加上英文的点，就会无意间产生列表，比如2017.12.30 这时候想表达的是日期，有些软件把它被误认为是列表。解决方式：在每个点前面加上\就可以了。

2017.12.30

表格

|学号|姓名|分数|

|:-|:-:|-:|

|1|小明|80|
